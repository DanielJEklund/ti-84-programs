INSTALLATION
Send the following to a Ti-84+CSE using software of 
your choice
-prgmYAHTCSE (YAHTCSE.8xp)
-prgmYZTXTOUT (YZTXTOUT.8xp)
-appVarYZSPTS (YZSPTS.8xv)


NOTE: This game requires the installation of DOORS CS for cse.
https://www.cemetech.net/programs/84pcse/asm/shells/dcse8.zip
----------------------

HOW TO PLAY
-The object of Yahtzee is to score maximum points.
-Rules located at:
http://www.yahtzee.org.uk/rules.html

----------------------

KEYS
-[2nd] acts as enter on any selection choice.
-[Y= - Graph] this row acts as the dice hold button. Clicking it will deselect the selected dice.
-[Clear] quits the game at any time.
-[Enter] rerolls the unselected dice.
-[Arrow keys] will move the cursor around the score card, as well as move letters up or down on name entry.
